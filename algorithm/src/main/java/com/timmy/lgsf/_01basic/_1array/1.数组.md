

#### 数组

- 数组是数据结构中的最基本结构，他可以用来存放若干个相同类型的数据元素
- 数组在内存中是连续存放的，数组内的数据，可以通过索引值直接取出得到。

##### 数组特点：

1. 数组容量固定不变，需在创建数组时指定数组大小
2. 使用连续的物理空间存取数据
3. 可以通过下标在O(1)的时间复杂度下读取数据
